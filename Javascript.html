Javascript

¿Qué es Javascript?

JavaScript es el único lenguaje de programación que funciona en los navegadores de forma nativa (lenguaje interpretado sin necesidad de compilación). Por tanto se utiliza como complemento de HTML y CSS para crear páginas webs.

¿Qué se puede hacer con JavaScript?

Ejemplo

Card Desplegable

Para hacer un card desplegable se debe de crear un <div> principal con la clase "card" como el siguiente:

<div class="card"></div>

Después, dentro del <div> se debe crear un <button>

<div class="card">

<button class="btn" id="btn1"> </button>

</div>

y un subcard <div>

<div class="card">

<button class="btn" id="btn1"> </button>

<div id="card-1" class="subcard">

</div>

</div>

el css :

.card{

margin-left: 5%;

background-color: whitesmoke;

width: 80%; padding: 20px 2.5% 20px 2.5%;

margin-top: 10px; }


.btn{

width: 30px;

background-color: whitesmoke; }

#card-1{

display: none; }


.subcard{

margin-top: 5px; }

el Javascript:

el Javascript:

function desplegarCard1() {

let card1 = document.getElementById("card-1");

if (card1.style.display === "none") {

card1.style.display = "block";

} else {

card1.style.display = "none"; }

}

document.getElementById('btn1').onclick = function (){

desplegarCard1(); }
