Javascript

What is Javascript?

JavaScript is the only programming language that works natively in browsers (interpreted language without the need for compilation). Therefore, it is used as a complement to HTML and CSS to create web pages.

What can be done with JavaScript?

Example

Pop-up Card

To create a folding card, you must create a tag <div> main with the class "card" like the following:

<div class="card"></div>

Then, inside the <div> you must create a <button>

<div class="card">

<button class="btn" id="btn1"> </button>

</div>

y un subcard <div>

<div class="card">

<button class="btn" id="btn1"> </button>

<div id="card-1" class="subcard">

</div>

</div>

The CSS:

.card{

margin-left: 5%;

background-color: whitesmoke;

width: 80%; padding: 20px 2.5% 20px 2.5%;

margin-top: 10px; }


.btn{

width: 30px;

background-color: whitesmoke; }

#card-1{

display: none; }


.subcard{

margin-top: 5px; }

The Javascript:

function desplegarCard1() {

let card1 = document.getElementById("card-1");

if (card1.style.display === "none") {

card1.style.display = "block";

} else {

card1.style.display = "none"; }

}

document.getElementById('btn1').onclick = function (){

desplegarCard1(); }
